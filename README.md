# Homebrew Tap for spotatui

This is the official Homebrew tap for [spotatui](https://github.com/LargeModGames/spotatui), a Spotify client for the terminal written in Rust.

## Installation

```bash
brew tap LargeModGames/spotatui
brew install spotatui
```

Or install directly:

```bash
brew install LargeModGames/spotatui/spotatui
```

## Updating

```bash
brew update
brew upgrade spotatui
```

## Uninstalling

```bash
brew uninstall spotatui
brew untap LargeModGames/spotatui  # Optional: remove the tap
```

## Requirements

- macOS (Intel or Apple Silicon) or Linux (x86_64)
- Spotify Premium account (required for native streaming)
